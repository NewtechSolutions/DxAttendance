设置用户一体化模板，即将软件一体化模板数据发送到机器内。

参数
参数说明如下表：
name type param direction description of param
dwMachineNumber LONG [in] 机器号
TableName BSTR [in] 表名
Datas BSTR [in] 数据记录
Options BSTR [in] 扩展使用，暂时没用，默认为空
返回
返回值说明如下：
name type description of value
True BOOL 函数执行成功
False BOOL 函数执行失败
参见
注意
TableName：仅支持一体化模板表名:pers_biotemplate
Datas：数据是文本格式的，第一条记录为需要设置的表字段，后面记录为需要设置的字段数据，多条记录之间用\r\n 分隔，各个“字段=值”对之间用\t 分隔；
Type=1 表示指纹
Type=2 表示面部

Pers_BioTemplate 表结构:
Field Value Type annotations
id Not Null int4 自增长字段
person_id Not Null int4 外键,用户ID
create_operator varchar(30) 创建人
create_time datatime 创建时间(2015-06-11 09:48:11)
valid_flag Not Null char(1) 是否有效标示，0：无效，1：有效，默认为1
is_duress Not Null char(1) 是否胁迫标示，0：非胁迫，1：胁迫，默认为0
bio_type Not Null int2 物识别类型，0：通用的 1：指纹 2：面部 3：声纹 4：虹膜 5：视网膜 6：掌纹 7：指静脉 8：掌静脉
major_ver Not Null char(1) 指纹算法版本有：9.0和10.3 面部算法版本有：5.0、7.0和8.0 指静脉算法版本有：3.0 掌静脉算法版本有：1.0 如：指纹算法版本10.3，主版本是10，副版本是3
minor_ver Not Null char(1) 指纹算法版本有：9.0和10.3 面部算法版本有：5.0、7.0和8.0 指静脉算法版本有：3.0 掌静脉算法版本有：1.0 如：指纹算法版本10.3，主版本是10，副版本是3
data_format int2 模板格式，如指纹有ZK等格式0：ZK 1：ISO 2：ANSI
template_id Not Null int4 外键(FK)，模板数据ID
template_no Not Null int2 生物具体个体编号，【指纹】编号是：0-9，对应的手指是：左手：小拇指/无名指/中指/食指/拇指, 右手：拇指/食指/中指/无名指/小拇指；【指静脉】 和指纹相同【面部】 template_no = 0【虹膜】 template_no按照0为左眼 1为右眼【掌静脉】 template_no按照0为左手 1为右手如果template_no没有明确定义，则默认值为0。
template_no_index Not Null int2 生物具体个体模板编号，如1个手指存储多枚模板。从0开始计算

获取用户一体化模板，即将机器内一体化模板拉取到软件上

参数
参数说明如下表：
name type param direction description of param
dwMachineNumber LONG [in] 机器号
Buffer BSTR [out] 缓存用于存储一体化模板信息
BufferSize LONG [in] 缓存大小
TableName BSTR [in] 一体化模板表名
FiledNames BSTR [in] 字段名列表
Filter BSTR [in] 读取数据条件
Options BSTR [in] 参数
返回
返回值说明如下：
name type description of value
True BOOL 函数执行成功
False BOOL 函数执行失败
参见
注意
Buffer 用于接收返回数据的缓冲区，返回的数据是文本格式的，可能是多条记录，各条记录之间用\r\n 分隔。
BufferSize 用于接收返回数据的缓冲区大小。
TableName 只支持pers_biotemplate（一体化），不支持user，固件不支持
FiledNames 字段名列表，多个字段之间用 分开，“*”表示全部字段，此时返回数据字段的第一行是字段名。
Filter 读取数据的条件，当单独一个“字段名操作符值”构成的字符串时，可以支持多个条件，使用逗号分隔，按如下方式：< 字段名>="">=< 值>="">（“=”符号两边不可以有空格）
Type=1 表示指纹
Type=2 表示面部
Options 当前仅在下载门禁事件记录表的数据时有效，值为“NewRecord”时下载新记录，当为空时下载全部记录。下载其他表数据时，该字段置为空字符串即可。
注解
适用于BW,TFT设备及新架构机器

◆ BiometricType

BiometricType

BiometricType

参数
BiometricType 获取支持生物识别类型。通过获取该参数返回8位数字字符串,每一位代表一种生物识别类型（0表示不支持；1表示支持;例如:BiometricType=01100000 ,表示设备支持指纹和面部）
字符索引 0 1 2 3 4 5 6 7 8
代表类型 通用 指纹 面部 声纹 虹膜 视网膜 掌纹 指静脉 掌静脉
参见
注意
注解
用于模板一体化接口

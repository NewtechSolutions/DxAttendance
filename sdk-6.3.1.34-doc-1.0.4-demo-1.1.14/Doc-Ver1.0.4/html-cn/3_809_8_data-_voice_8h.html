◆ EnableCustomizeVoice()

VARIANT_BOOL EnableCustomizeVoice ( LONG  dwMachineNumber,
LONG  VoiceID,
LONG  Enablel 
)

是否启用指定序号声音的自定义声音功能

参数
参数说明如下表：
name type param direction description of param
dwMachineNumber LONG [in] 机器号
VoiceID LONG [in] 声音序号
Enablel LONG [in] 是否启用定序号声音的自定义声音功能，1为启用，0为禁用
返回
返回值说明如下：
name type description of value
True BOOL 函数执行成功
False BOOL 函数执行失败
参见
注意
1.该功能为定制功能，使用该功能机器需有自定义声音功能。
2.自定义声音功能描述：当前机器在执行一些操作时会播放相应语音，利用该功能可使机器播放自定义语音，如当机器验证通过后会播放序号为1的语音，即”thank you”,使用该功能后，可使机器在验证通过后播放自定义声音。
如：EnableCustomizeVoice（1,1,1)//启用1号语音的自定义功能。SetCustomizeVoice(1,1,‘C:\test.wav’)//设置1号语音的自定义语音功能。
注解
适用于BW设备

name type param direction description of param
dwMachineNumber LONG [in] 机器设备号
dwEnrollNumber BSTR [in] 用户号

返回
返回值说明如下：
name type description of value
True BOOL 函数执行成功
False BOOL 函数执行失败
参见
注意
1.VerifyStyle参数，用于表示验证方式，取值分别如下：
Group Verify = 0
FP/PW/RF = 128
FP = 129
PIN = 130
PW = 131
RF = 132
FP/PW = 133
FP/RF = 134
PW/RF = 135
PIN&FP = 136
FP&PW = 137
FP&RF = 138
PW&RF = 139
FP&PW&RF = 140
PIN&FP&PW = 141
FP&RF/PIN = 142
2.Reserved参数保留，目前不用。
注解
该接口目前适用于新架构固件。

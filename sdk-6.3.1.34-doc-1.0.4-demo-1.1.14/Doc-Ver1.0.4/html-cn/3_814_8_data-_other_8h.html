◆ SetEventMode()

VARIANT_BOOL SetEventMode ( LONG  newVal)

设置实时事件模式

参数
参数说明如下表：
name type param direction description of param
newVal LONG [in] 属性值
返回
返回值说明如下：
name type description of value
True BOOL 函数执行成功
False BOOL 函数执行失败
参见
注意
1.newVal为1，默认走原有流程，即sdk内部创建窗体，通过窗口消息转发实时事件。要求客户必须在UI线程创建ActiveX对象。
2.newVal为0，SDK内部使用线程转发实时事件(现有方式)，对于ActiveX创建无要求，由于工作线程无法直接操作用户界面，因此应用层需在处理实时事件转发消息到界面线程。
注解
该接口目前适用于新架构固件。
